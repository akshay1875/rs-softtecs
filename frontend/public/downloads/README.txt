RS Softtecs - Download Files
============================

Please add the following PDF files to this folder:

1. RS-Softtecs-Brochure.pdf - Company brochure with all course details
2. RS-Softtecs-Syllabus.pdf - Detailed course syllabus

These files will be downloaded when users click the "Download Brochure" 
or "Download Syllabus" buttons on the website.

If files are not present, the download will fail gracefully and 
the enquiry popup will still appear to capture leads.
